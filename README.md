# debian-13-rdp-xrdp-auto
# 🖥️ Debian 13 XRDP + XFCE + Root Login (Quick Setup)

This script installs a **full XFCE desktop with XRDP** on Debian 13 (Trixie) and enables **root login via RDP**.

---

## ⚡ Features
- XFCE desktop environment
- XRDP server (RDP access)
- Root login enabled
- Fixes common Xorg/XRDP issues on Debian 13

---

## 🚀 Installation

```bash
chmod +x install.sh
sudo ./install.sh
```

---

## ⚠️ Notes
This setup is intended for Debian 13
Root login via RDP is not recommended for production! USE WITH EXTREM CAUTION!
If exposed to the internet → DON'T USE IT.
